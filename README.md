# Whisk Prompt Automator

Uma extensão para automatizar o envio de múltiplos prompts para o gerador de imagens **Google Whisk**, agilizando a criação de imagens em lote.

## Funcionalidades

- **Envio em Lote:** Insira uma lista de prompts e a extensão os enviará um por um.
- **Download Automático com Organização:** Baixe automaticamente todas as imagens geradas. Você pode escolher um nome de pasta específico para que a extensão salve tudo organizado dentro do seu diretório "Downloads".
- **Aspect Ratio Randômico:** Gere imagens com proporções variadas (16:9, 1:1, 3:4, etc.) de forma automática. A extensão pode escolher uma proporção aleatória para cada novo prompt a partir de uma lista que você define.
- **Controle de Delay:** Configure um atraso (em segundos) entre cada envio para controlar o ritmo.
- **Persistência de Dados:** Suas listas de prompts e configurações ficam salvas localmente, prontas para a próxima vez que você usar.
- **Status em Tempo Real:** Acompanhe o progresso da automação diretamente na janela da extensão.

## Como Instalar

Como esta é uma extensão em desenvolvimento, ela precisa ser instalada manualmente no modo de desenvolvedor do seu navegador (Chrome, Edge, etc.).

1.  **Faça o download:** Baixe os arquivos da extensão e descompacte-os em uma pasta no seu computador.
2.  **Acesse as Extensões:**
    -   No Google Chrome, acesse `chrome://extensions`
    -   No Microsoft Edge, acesse `edge://extensions`
3.  **Ative o Modo de Desenvolvedor:** Procure por um interruptor chamado "Modo de desenvolvedor" (ou "Developer mode") e ative-o. Geralmente fica no canto superior ou inferior direito da página.
4.  **Carregue a Extensão:**
    -   Clique no botão **"Carregar sem compactação"** (ou "Load unpacked").
    -   Uma janela para selecionar uma pasta irá abrir.
    -   Navegue até a pasta onde você descompactou os arquivos da extensão e selecione-a.
5.  **Pronto!** A extensão "Whisk Prompt Automator" aparecerá na sua lista de extensões e estará pronta para ser usada na página do Google Whisk.
